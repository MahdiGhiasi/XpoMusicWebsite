Xpo Music Keyboard Shortcuts

Xpo Music v2.0+ supports keyboard shortcuts, so you can navigate and control Xpo Music while keeping your hands on the keyboard!

Here are the keyboard shortcuts currently supported by Xpo Music:

Ctrl+C Copy current page URL
Ctrl+V Open the Spotify link from clipboard
Space Play/Pause
Ctrl+Right Next Track
Ctrl+Left Previous Track
Ctrl+Up Volume Up
Ctrl+Down Volume Down
Ctrl+Shift+Up Max Volume
Ctrl+Shift+Down Mute
Ctrl+L Search
Ctrl+N Create New Playlist
Alt+Left Back
Alt+Right Forward
Up Select previous item in the track list
Down Select next item in the track list
Left Select previous item in the track list
Right Select next item in the track list
Ctrl+P Settings
Ctrl+M Mini View
Ctrl+, Now Playing
Ctrl+Q Queue
Alt+1 Home
Alt+2 Search
Alt+3 Your Library
Alt+4 Now Playing
Esc Exit Now Playing or Mini View modes
Alt+F4 Close Xpo Music
